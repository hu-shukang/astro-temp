# 電力会社ユーザーポータル - UI/UX 設計

## 1. デザインシステム

### スタイル

**Flat Design** — 2D・ミニマリスト・クリーンな線・タイポグラフィ中心
電力ポータルのような公共インフラ系サービスに最適。信頼感・操作性・WCAG AAA 準拠を重視。

### カラーパレット

| 役割       | Hex       | 用途                             |
| ---------- | --------- | -------------------------------- |
| Primary    | `#2563EB` | ボタン・アクティブリンク・強調   |
| Secondary  | `#3B82F6` | ホバー・サブアクション           |
| CTA        | `#F97316` | 重要アクション（申請・ポイント） |
| Background | `#F8FAFC` | ページ背景                       |
| Surface    | `#FFFFFF` | カード・モーダル背景             |
| Text       | `#1E293B` | 本文                             |
| Muted      | `#475569` | サブテキスト・ラベル             |
| Border     | `#E2E8F0` | 区切り線・カード境界             |
| Success    | `#10B981` | 支払済・完了                     |
| Warning    | `#F59E0B` | 期限間近・注意                   |
| Danger     | `#EF4444` | 未払い・エラー                   |

### タイポグラフィ

- **見出し**: Fira Code（数値・金額表示に最適）
- **本文**: Fira Sans（可読性・多言語対応）

```css
@import url("https://fonts.googleapis.com/css2?family=Fira+Code:wght@400;500;600;700&family=Fira+Sans:wght@300;400;500;600;700&display=swap");
```

| 要素              | フォント  | サイズ  | Weight |
| ----------------- | --------- | ------- | ------ |
| ページタイトル    | Fira Code | 24px    | 600    |
| セクション見出し  | Fira Sans | 18px    | 600    |
| 本文              | Fira Sans | 16px    | 400    |
| ラベル            | Fira Sans | 14px    | 500    |
| 数値（金額・kWh） | Fira Code | 20-32px | 700    |
| キャプション      | Fira Sans | 12px    | 400    |

### 3-2. 電力使用量 (`/electricity`)

月別・日別を **Segmented Controls** で切り替える1ページ構成。

```
┌──────────────────────────────────────────────────────┐
│ [ContractSelector ▼]                                 │
├──────────────────────────────────────────────────────┤
│         [ 月別 | 日別 ]  ← Segmented Controls        │
├──────────────────────────────────────────────────────┤
│ ▼ 月別表示時                                         │
│ 月別使用量グラフ（過去12ヶ月）                        │
│ ┌─────── ComposedChart (Bar + Line) ─────────────┐   │
│ │  kWh                                           │   │
│ │  400 ┤  ████         ●━━━━●━━━━●              │   │
│ │  300 ┤  ████ ████ ●━━             ●━━...       │   │
│ │  200 ┤  ████ ████ ████ ...                     │   │
│ │      └──────────────────── 月                  │   │
│ │  ■ 今年（棒）  ●━━ 前年（折線）                │   │
│ └────────────────────────────────────────────────┘   │
│ 今月の予測: 現在 320 kWh → 月末予測 420 kWh          │
│ 予測電気代: ¥10,800                                  │
│ 前年同月比 YoYCard: 320 kWh vs 290 kWh  ▲+10.3%     │
├──────────────────────────────────────────────────────┤
│ ▼ 日別表示時                                         │
│ [< 前月]  2025年3月  [次月 >]                        │
│ ┌─────── ComposedChart (Bar + Line) ─────────────┐   │
│ │  1  2  3  4  5  ...  31 日                      │   │
│ │  ■ 今年（棒）  ●━━ 前年同月（折線）             │   │
│ └────────────────────────────────────────────────┘   │
│ 月間合計: 320 kWh  |  平均: 10.3 kWh/日              │
└──────────────────────────────────────────────────────┘
```

**Segmented Controls 仕様**:

- 2セグメント: 「月別」「日別」
- アクティブ: `bg-primary text-white rounded`、非アクティブ: `text-muted`
- 切り替え時: コンテンツエリアのみ差し替え（ページ遷移なし）
- URL: クエリパラメータで状態を保持（`/electricity?view=monthly` / `?view=daily`）

**チャート仕様**: `ComposedChart`（Recharts）

- 今年度: `Bar` — `#2563EB`（青）
- 前年度: `Line` — `#F97316`（オレンジ）、`strokeWidth: 2`、ドットあり（`dot: true`）

**Tooltip 仕様**:

```
┌──────────────────────────┐
│ 2025年3月                 │
│ ■ 今年  320 kWh           │
│ ● 前年  290 kWh           │
│ 差分    +30 kWh (+10.3%)  │
└──────────────────────────┘
```

- ホバーで表示、カーソル位置に追従
- `cursor: crosshair` でデータ点を指示
- カスタム `<Tooltip content={...} />` で差分も表示

**日別・月ナビゲーション**: 翌月が未来の場合は「次月」ボタン disabled

### 3-5. お知らせ (`/notifications`)

```
┌──────────────────────────────────────────────────────┐
│ お知らせ  [一括既読にする]                           │
├──────────────────────────────────────────────────────┤
│ ● 請求書が発行されました       2025/03/01           │
│   2025年3月分・¥8,420 [詳細を見る →]               │
├──────────────────────────────────────────────────────┤
│ ─ キャンペーンのお知らせ       2025/02/15           │
│   夏の節電キャンペーン開始のお知らせ                │
└──────────────────────────────────────────────────────┘
```

**タイトル（共通）と本文（個別）の分離**:

| 種別              | タイトル（共通・固定）             | 本文（個別データ）         |
| ----------------- | ---------------------------------- | -------------------------- |
| `bill_issued`     | 請求書が発行されました             | 対象月・金額               |
| `bill_due`        | お支払期限が近づいています         | 期限日・金額               |
| `usage_alert`     | 電気使用量が増加しています         | 増加率・使用量             |
| `contract_update` | 申請が完了しました                 | 申請種別                   |
| `points_expiring` | ポイントの有効期限が近づいています | 失効予定ポイント数・失効日 |
| `campaign`        | キャンペーンのお知らせ             | キャンペーン名・期間       |
| `system`          | メンテナンスのお知らせ             | 実施日時                   |

未読: 左に青い丸アイコン / 既読: グレー
クリックで詳細展開（アコーディオン）または詳細モーダル

- ドロップダウン: ホバーまたはクリックで展開、`shadow-md rounded-md` のパネル
- 未読バッジ: お知らせリンクの右に赤丸数字

## 5. インタラクション設計

### アニメーション

| 要素           | アニメーション                 | 時間  |
| -------------- | ------------------------------ | ----- |
| ページ遷移     | fade-in（opacity 0→1）         | 150ms |
| モーダル表示   | scale 0.95→1 + opacity         | 200ms |
| ドロップダウン | slide-down + fade              | 150ms |
| ホバーボタン   | color/shadow shift             | 150ms |
| チャートロード | 棒が下から伸びるアニメーション | 500ms |

`prefers-reduced-motion` が有効な場合: アニメーションを無効化

### ローディング状態

| シーン             | 表示方法                   |
| ------------------ | -------------------------- |
| 契約切り替え       | コンテンツエリアにスピナー |
| チャートデータ取得 | Skeleton（棒グラフ形状）   |
| フォーム送信中     | ボタン disabled + スピナー |
| ページ初回ロード   | Skeleton カード            |

### エラー状態

| シーン                 | 表示方法                          |
| ---------------------- | --------------------------------- |
| フォームバリデーション | 入力欄下に赤テキスト              |
| API エラー             | Toast 通知（右上・3秒で消える）   |
| 空データ               | Empty state イラスト + メッセージ |

- ContractSummaryCard: 縦並び
- チャート: 横スクロール可

## 7. アクセシビリティ

- 最小フォントサイズ: 16px（モバイル本文）
- コントラスト比: 本文 4.5:1以上（WCAG AA）
- フォーカスリング: `outline-2 outline-primary outline-offset-2`
- タッチターゲット: 最小 44×44px
- フォーム: `<label for="...">` を必ず付与
- カラーのみで状態を示さない（アイコン・テキスト併用）
- キーボードナビゲーション: Tab 順序はビジュアル順
